# t-rkiye-havalimanlar--i-in-veri-zarflama-analizi
Bu çalışmada, Türkiye'deki havaalanlarının fiziksel altyapı kapasiteleri (pistler, terminaller, apronlar) ve işletme giderleri girdi olarak, yolcu/kargo trafiği hacimleri ve finansal gelirleri ise çıktı olarak modellenmiş ve Veri Zarflama Analizi (DEA) kullanılarak göreceli performans ve verimlilik ölçümleri yapılmıştır.
